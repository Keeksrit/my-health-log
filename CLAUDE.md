# Health Log

## Design Context

This project has impeccable design context. Before any UI work, read:

- **PRODUCT.md** — register (product), platform (web), users, purpose, brand personality, anti-references, and design principles.
- **DESIGN.md** — the visual system: palette, typography, elevation, components, and do's/don'ts (`.impeccable/design.json` is the machine-readable sidecar).

Register: **product** · Platform: **web** (mobile web PWA, 430px hand-held column).

Personality in one line: **calm but rigorous, humane** — a warm "Lab Notebook", never a boring or sterile clinical EMR.

Non-negotiables: keep the cream canvas + single forest-green accent; back every color-coded signal (severity, food type, status) with a second cue for color-blind safety; meet WCAG AA contrast; reserve Lora for headings only.
